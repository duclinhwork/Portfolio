# Project Brief - Portfolio Website

## Project Overview
This is a professional portfolio website for Nguyễn Đức Linh, a Data Analyst and Machine Learning Engineer. The website serves as a digital CV and portfolio showcasing professional experience, skills, projects, and certifications.

## Core Requirements
- **Bilingual Support**: Vietnamese and English versions
- **Professional Presentation**: Clean, modern design suitable for professional networking
- **Responsive Design**: Mobile-first approach for accessibility across devices
- **Easy Content Management**: JSON-based data structure for easy updates
- **Automated Deployment**: GitHub Pages deployment with GitHub Actions

## Technology Stack
- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS with custom design system
- **Animations**: Framer Motion for smooth interactions
- **Icons**: Lucide React
- **Deployment**: GitHub Pages with automated CI/CD
- **Data Management**: JSON files for content
